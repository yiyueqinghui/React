初始化webpack npm init
安装webpack 和 webpack-cli npm install webpack webpack-cli --save--dev 会产生node_modules文件夹，并将这两个依赖加入其中，同时也写入package.json中。 之后在工程中就可使用webpack命令了
在package.json中的scripts中配制命令，例如："build": "webpack --config webpack-config.js" ，即执行配制文件webpack-config.js
编译jsx和es6语法为js,步骤如下:
  • 1,可采用以下依赖 npm install babel-loader @babel/core @babel/preset-env -D 其中，babel-loader 转换j加载器，@babel/core 为babel的核心模块 ,@babel/preset-env 转换依据，将ES6转为ES5
  • 2,配制webpack.config.js中的modules==>rules
对css处理,将css 写入js：
  • 1,安装 npm install css-loader style-loader --save--dev
  • 配制rules
  • 下载依赖 npm install url-loader file-loader -D
  • 配制rules
对less sass 处理
  • npm install less less-loader -D
自动生成index.html
  • npm install html-webpack-plugin@next --save-dev
自动打开浏览器
  • npm install webpack-dev-server --save-dev
  • 新增命令 "dev": "webpack-dev-server --devtool eval --progress --colors --hot"
